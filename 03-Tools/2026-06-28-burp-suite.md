# Burp Suite

Integrated platform for web application security testing.

**Key Features:**
- Intercepting proxy
- Vulnerability scanner
- Repeater for manual testing
- Intruder for automated attacks

**Getting Started:**
1. Configure browser proxy to 127.0.0.1:8080
2. Install Burp CA certificate
3. Intercept requests in Proxy tab
4. Send to Repeater for testing
5. Use Intruder for automation
