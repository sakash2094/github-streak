# Nmap (Network Mapper)

Industry-standard network discovery and security auditing tool.

**Essential Commands:**
```bash
# Basic scan
nmap 192.168.1.1

# Service version detection
nmap -sV 192.168.1.1

# Aggressive scan
nmap -A 192.168.1.1

# All ports
nmap -p- 192.168.1.1

# Save output
nmap -oN scan.txt 192.168.1.1
```

**Use Cases:**
- Network inventory
- Service version detection
- Security auditing
- Port scanning
