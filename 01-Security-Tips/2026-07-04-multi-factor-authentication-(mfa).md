# Multi-Factor Authentication (MFA)

Enable MFA on all critical accounts. Use an authenticator app or hardware key instead of SMS.

**Why MFA Matters:**
- Adds an extra layer of security
- Prevents 99.9% of automated attacks
- Even if password is stolen, account remains secure

**Recommended Methods:**
1. Hardware keys (YubiKey)
2. Authenticator apps (Google Authenticator, Authy)
3. Avoid SMS when possible
